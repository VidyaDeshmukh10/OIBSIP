# Retail Sales EDA Dashboard (Power BI)

# Project Overview
This project presents an Exploratory Data Analysis (EDA) of retail sales data using Power BI to identify trends, patterns and insights.
The dashboard provides insights into sales performance across months, product categories, and gender.
# Key KPIs
- Total Sales (INR)
- Total Transactions
- Total Quantity Sold
- Average Transaction Value
# Dashboard Features
- Interactive slicers for Month, Product Category, and Gender
- Sales contribution by gender
- Total sales by product category
- Monthly sales trend analysis
# EDA Concepts Used
- Data cleaning aggregation
- Descriptive analysis using KPIs
- Trend analysis of monthly sales
- Category-wise and gender-wise comparison
- Interactive Filter-based analysis using slicers
# Files Included
- Power BI dashboard file (.pbix)
- Dashboard preview images
- Filter-applied analysis image
- Dashboard demo video (no voice)
# Tools Used
- Power BI
- Excel (data source)
# Insights
This dashboard helps identify sales trends ,high-performing product categories and customer behaviour patterns using interactive exploration.
