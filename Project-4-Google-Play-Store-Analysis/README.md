# Project Overview
This project focuses on analyzing Google Play Store data to understand app market dynamics, trends, and user behavior using *Python* and *Power BI*.

# Objectives
- Clean and preprocess Google Play Store datasets
- Analyze app categories, ratings, pricing, and installs
- Perform sentiment analysis on user reviews
- Build an interactive Power BI dashboard
- Gain hands-on experience in data visualization

# Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Power BI
- CSV Datasets

# Key Analysis Performed
- App distribution across categories
- Free vs Paid apps comparison
- Average app rating by category
- User reviews sentiment analysis
- Price distribution of paid apps

# Dashboard
The Power BI dashboard provides an interactive overview of:
- Total number of apps
- Average app ratings
- Category-wise app distribution
- Free vs Paid apps comparison
- User sentiment insights
